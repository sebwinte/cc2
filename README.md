### Settings:

```
{   
    "compression":[
        {
            "low":30,
            "medium": 60,
            "high": 90
        }
    ],
    "settings":[
        {
            "notification": 0,
            "cc2_folder" : "testordner/"
        }
    ]
}
```

Params

| param             | default       |
| -------------     |:-------------:|
| low               | 30%           | 
| medium            | 60%           |   
| high              | 90%           |   

Feel free to change 

* Notification: 0/1 Turn the notifications on or off.
* cc2_folder: Declare the folder you want to be watched over ..
